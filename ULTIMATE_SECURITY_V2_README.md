### **Interface Guard in Code verwenden:**
```python
from ava.security.interface_guard import get_interface_guard, InterfaceType

guard = get_interface_guard()

# Interface registrieren
interface_id = guard.register_interface(
    InterfaceType.HTTP, 
    "0.0.0.0", 
    8080,
    description="My API",
    whitelist_only=True  # Nur Whitelist erlaubt
)

# Zugriff prüfen
result = guard.check_access(
    interface_id, 
    "192.168.1.100",  # Source IP
    12345,            # Source Port
    payload="GET /api/users",
    threat_score=0.5
)

if result['action'] == 'ANNIHILATE':
    print(f"💥 THREAT ANNIHILATED: {result['reason']}")
```
